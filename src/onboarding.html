Welcome to Text-to-Speech Extension!

Thank you for installing our extension. This guide will help you get started and make the most of the text-to-speech features.

Getting Started

1. Setting Up Your API Key

  1. Go to the extension options by clicking the extension icon and selecting "Options"
  2. Click on "Create API Key" or enter your existing API key
  3. Save your settings

How to Use

  1. Select any text on a webpage
  2. Click the extension icon or use the keyboard shortcut
  3. The selected text will be read aloud
Tip: You can customize voice settings and other preferences in the extension options.

Keyboard Shortcuts

  • Ctrl/Cmd + Shift + S - Start/Stop reading
  • Ctrl/Cmd + Shift + P - Pause/Resume
